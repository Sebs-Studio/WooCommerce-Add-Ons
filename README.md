== Description ==

WooCommerce Add-Ons is an open community plugin to add small pieces of code that offer new filters, options and abilities.

The plugin is designed to apply additional functions that have not been added in the core of WooCommerce and to expand on what’s already there.

It's also a great plugin to help developers generate custom settings for their own WooCommerce plugin.

== Installation ==

= Minimum Requirements =

* WooCommerce 1.6.2 or higher.

= Manual installation =

The manual installation method involves downloading the plugin and uploading it to your webserver via your favourite FTP application.

1. Download the plugin file to your computer and unzip it
2. Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installation's wp-content/plugins/ directory.
3. Activate the plugin from the Plugins menu within the WordPress admin.

= Upgrading =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
